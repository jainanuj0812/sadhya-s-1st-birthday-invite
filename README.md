## ✨ Features

- **Beautiful Barbie Theme**: Pink, magenta, and gold color palette with custom SVG Barbie elements
- **Interactive Design**: Floating animations, sparkles, and magical effects
- **PDF Download**: Generate and download a high-quality PDF version of the invitation
- **Responsive**: Looks great on all devices
- **Countdown Timer**: Real-time countdown to the party
- **Interactive Map**: Embedded Google Maps for venue location
- **RSVP Integration**: WhatsApp RSVP functionality
- **Custom Barbie Elements**: 
  - Barbie crowns, dresses, shoes, bags
  - Magic wands, castles, bows, hearts
  - Beautiful Barbie illustration
  - Floating decorative elements

## 🚀 Getting Started

### Local Development
```bash
# Clone the repository
git clone https://github.com/jainanuj0812/sadhya-s-1st-birthday-invite.git
cd sadhya-s-1st-birthday-invite

# Install dependencies
npm install

# Start development server
npm run dev
```

### Deployment to GitHub Pages

This project is configured for automatic deployment to GitHub Pages:

1. **Push to GitHub**: Push your changes to the main branch
2. **Automatic Build**: GitHub Actions will automatically build and deploy
3. **Live URL**: Visit https://jainanuj0812.github.io/sadhya-s-1st-birthday-invite/

#### Manual Deployment
```bash
# Build the project
npm run build

# Or use the deployment script
./deploy.sh
```
