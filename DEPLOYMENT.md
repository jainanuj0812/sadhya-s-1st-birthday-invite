## 📝 Steps to Deploy

### 1. Create GitHub Repository
```bash
# If you haven't already, create the repository on GitHub
# Repository name should be: sadhya-s-1st-birthday-invite
```

### 2. Push Code to GitHub
```bash
# Initialize git (if not already done)
git init

# Add all files
git add .

# Commit changes
git commit -m "🎀 Add Sadhya's Barbie-themed birthday invitation with PDF download"

# Add GitHub remote
git remote add origin https://github.com/jainanuj0812/sadhya-s-1st-birthday-invite.git

# Push to main branch
git push -u origin main
```

### 3. Enable GitHub Pages
1. Go to your GitHub repository: https://github.com/jainanuj0812/sadhya-s-1st-birthday-invite
2. Click on **Settings** tab
3. Scroll down to **Pages** section (in left sidebar)
4. Under **Source**, select **"GitHub Actions"**
5. The deployment will start automatically!

### 4. Wait for Deployment
- GitHub Actions will automatically build and deploy your site
- Check the **Actions** tab to see the deployment progress
- Once complete (green checkmark), your site will be live!
